# WTN Demo

WebRTC 传输网络 Demo

## 运行项目

```Shell
yarn install

yarn dev
```

## 项目结构

```shell
|-- Project
  |-- assets 图片、字体等资源
  |-- src 源代码
    |-- app 请求封装
    |-- components Demo中复用的组件
    |-- lib WebRTC的封装
	|-- locales 多语言
    |-- pages 页面
    |-- store 业务数据模型(redux)
	|-- utils 公共函数
    |-- config.ts 静态常量

```
